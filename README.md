# Object-Detection-using-YOLO
A YOLOv8-based object detection system trained on a COCO subset. Features an end-to-end pipeline for training, evaluation (51% mAP), and real-time visualization of multi-class detections.

## Features
- Trained on a subset of COCO dataset
- Detects multiple classes (car, airplane, truck, motorcycle, bus)
- Real-time object detection capabilities

## Model Performance
- Precision: 0.6628
- Recall: 0.4556
- mAP@0.5: 0.5142

## Tech Stack
- Python
- Ultralytics YOLOv8
- OpenCV
